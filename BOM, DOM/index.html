Task 7

Create an HTML page with a list of books. When clicking on an element, the text color should change to orange. When clicking on another book, the previous one should revert to its original color. If the Ctrl key is held while clicking, the element should be added to/removed from the selected ones. If the Shift key is held while clicking, all elements between the previously selected and the currently clicked ones should be selected.

Disable the default behavior for these key combinations.
